# Fcus

This is a minimal Markdown UWP editor made based on [Live (GitHub-Flavored) Markdown Editor](https://jbt.github.io/markdown-editor/).

### Usage:

- Ctrl-O:Open Exist File
- Ctrl-N:Open New File
- Ctrl-S:Save File
- Ctrl-A:About
